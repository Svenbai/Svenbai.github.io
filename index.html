git操作步骤

1.cd Desktop进入到桌面

2.mkdir work创建work文件夹

3.cd work进入到work文件夹

4.touch git命令.html新建一个HTML文件

5.在HTML中写一些内容

6.git init 初始化仓库

7.git config --global user.name "name"
git config --global user.email "@email"创建用户名和邮箱

8.git add . 将work中的所有的文件添加到仓库中

9.git commit -m "git操作步骤" 将文件提交到仓库(-m后的内容为注释)

10.git branch 进入到主线

11.在github中新建一个名为github用户名.github.io

12. git remote add origin https://github.com/GaoJingJing0102/GaoJingJing0102.github.io.git 本地仓库与远程仓库的链接

13.git push -u origin master 上传成功

14.通过github用户名.github.io可以进行访问
